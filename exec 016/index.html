Existem cinco valores de posiçoes diferentes:

  • static.
  • relative
  • fixed
  • absolute
  • stikey

Os elementos sao posicionados usando as propriedades top, bottom, left, right.

Position 'static'

Os elementos em html por padrão já são posicionados em static. Não são afetados pelas propriedades, top, left, right e bottom.

Position 'relative'

São afetados pelas propriedades top, left, right, e bottom. O conteudo não vai se ajustar para se encaixar no espaço deixado pelo elemento.

Exemplo

Position 'fixed'

O elemento é posicionado em relação a viewport, ou seja, ele vai permanecer no mesmo lugar da págna.
